## FreeAlign 核心算法详解: 位姿图优化 (Pose Graph Optimization)

   ┌─────────────────────────────────────────────────────────────────────────────────┐
   │              box_alignment_relative_sample_np 算法流程                          │
   │                D:\FreeAlign\opencood\models\sub_modules\box_align_v2.py          │
   └─────────────────────────────────────────────────────────────────────────────────┘

   输入:
   ├── pred_corners_list: 各 CAV 的检测框角点 [[N1, 8, 3], [N2, 8, 3], ...]
   ├── noisy_lidar_pose: 初始位姿 (可能为零或带噪声) [N_cav, 6]
   ├── uncertainty_list: 检测框不确定性权重 [[N1, 3], [N2, 3], ...]
   └── thres: 检测框匹配阈值 (默认 1.5m)

   输出:
   └── refined_pose: 校正后的相对位姿 [N_cav, 3] (x, y, yaw)

   ┌─────────────────────────────────────────────────────────────────────────────────┐
   │                        Step 1: 检测框坐标转换                                   │
   ├─────────────────────────────────────────────────────────────────────────────────┤
   │  第68-85行:                                                                       │
   │  │                                                                               │
   │  ├── 将检测框从各 CAV 本地坐标系转换到世界坐标系                                   │
   │  │   lidar_pose_noisy_tfm = pose_to_tfm(noisy_lidar_pose)                       │
   │  │   pred_corners_world = project_box3d(pred_corners, T_world_ego)              │
   │  │                                                                               │
   │  ├── 提取检测框属性                                                               │
   │  │   pred_box3d = corner_to_center(pred_corners)  # [x, y, z, w, l, h, yaw]    │
   │  │   pred_center_world = pred_box3d[:, :3]       # 检测框中心                    │
   │  │   pred_yaw_world = pred_box3d[:, 6]           # 检测框朝向                    │
   │  │                                                                               │
   │  └── 处理不确定性权重                                                             │
   │      pred_certainty = exp(-pred_log_sigma2)      # 不确定性 → 确定性权重         │
   │      pred_certainty[:,:2] /= d_a_square          # 归一化位置权重                 │
   └─────────────────────────────────────────────────────────────────────────────────┘
                                       │
                                       ▼
   ┌─────────────────────────────────────────────────────────────────────────────────┐
   │                        Step 2: 检测框配对与聚类                                   │
   ├─────────────────────────────────────────────────────────────────────────────────┤
   │  第90-165行: 核心创新点 - 基于检测框的空间一致性约束                                │
   │  │                                                                               │
   │  ├── 计算所有检测框中心之间的两两 L2 距离                                         │
   │  │   pred_center_allpair_dist = all_pair_l2(center_A, center_B)                 │
   │  │   └── 输出: [N_total_boxes, N_total_boxes] 距离矩阵                           │
   │  │                                                                               │
   │  ├── 聚类算法: 找出同一物体被多个 CAV 检测到的检测框对                              │
   │  │   for box_idx in range(cum):                                                  │
   │  │       # 找出距离小于阈值的检测框                                                │
   │  │       within_thres_idx = (dist[box_idx] < thres).nonzero()                   │
   │  │                                                                               │
   │  │       # 广度优先搜索扩展聚类                                                    │
   │  │       while unexplored:                                                       │
   │  │           idx = unexplored[0]                                                 │
   │  │           for newidx in neighbors:                                            │
   │  │               if newidx not in explored:                                      │
   │  │                   unexplored.append(newidx)                                   │
   │  │                                                                               │
   │  │       # 创建聚类记录                                                           │
   │  │       cluster_dict[cluster_id] = {                                            │
   │  │           'box_idx': [检测框索引列表],                                         │
   │  │           'box_center_world': [检测框中心],                                    │
   │  │           'box_yaw': [检测框朝向],                                             │
   │  │           'landmark': [x, y, yaw]  # 聚类中心作为地标                          │
   │  │       }                                                                        │
   │  │                                                                               │
   │  └── 关键约束条件                                                                 │
   │      ├── 同一 CAV 的检测框之间不建立连接 (设为 MAX_DIST)                          │
   │      ├── 只有不同 CAV 观测到的检测框才能配对                                      │
   │      └── 配对成功意味着两个 CAV 观测到了同一个物体                                │
   └─────────────────────────────────────────────────────────────────────────────────┘
                                       │
                                       ▼
   ┌─────────────────────────────────────────────────────────────────────────────────┐
   │                        Step 3: 构建位姿图 (Pose Graph)                           │
   ├─────────────────────────────────────────────────────────────────────────────────┤
   │  第180-220行:                                                                     │
   │  │                                                                               │
   │  位姿图结构:                                                                       │
   │  ┌─────────────────────────────────────────────────────────────┐                │
   │  │     Agent 0 (Ego, 固定)                                      │                │
   │  │         │                                                     │                │
   │  │         ├── Edge 1 ──── Landmark 1 (Object A)                 │                │
   │  │         │              (检测框聚类中心)                        │                │
   │  │         │                    │                                │                │
   │  │         │                    ├── Edge 2 ──── Agent 1          │                │
   │  │         │                    │                                │                │
   │  │         │                    └── Edge 3 ──── Agent 2          │                │
   │  │         │                                                     │                │
  │  │         └── Edge 4 ──── Landmark 2 (Object B)                 │                │
  │  │                              │                                │                │
  │  │                              └── Edge 5 ──── Agent 1          │                │
  │  └─────────────────────────────────────────────────────────────┘                │
  │  │                                                                               │
  │  ├── 添加 Agent 节点 (待优化的位姿)                                                │
  │  │   for agent_id in range(N):                                                   │
  │  │       pose = noisy_lidar_pose[agent_id, [0,1,4]]  # [x, y, yaw]              │
  │  │       if agent_id == 0:  # Ego 车固定为参考系                                   │
  │  │           pgo.add_vertex(id=agent_id, pose=pose, fixed=True)                 │
  │  │       else:                                                                   │
  │  │           pgo.add_vertex(id=agent_id, pose=pose, fixed=False)                │
  │  │                                                                               │
  │  ├── 添加 Landmark 节点 (检测框聚类中心)                                           │
  │  │   for landmark_id in range(N, cluster_id):                                    │
  │  │       landmark = cluster_dict[landmark_id]['landmark']  # [x, y, yaw]        │
  │  │       pgo.add_vertex(id=landmark_id, pose=landmark, fixed=False)             │
  │  │                                                                               │
  │  └── 添加 Agent-Landmark 边 (观测约束)                                             │
  │      for landmark in cluster_dict:                                               │
  │          for box_idx in landmark['box_idx']:                                     │
  │              agent_id = box_idx_to_agent[box_idx]                                │
  │              measurement = pred_box3d[box_idx]  # Agent 观测到的检测框位置        │
  │              information = pred_certainty[box_idx]  # 不确定性权重矩阵            │
  │              pgo.add_edge(                                                       │
  │                  vertices=[agent_id, landmark_id],                               │
  │                  measurement=measurement,                                        │
  │                  information=information                                         │
  │              )                                                                   │
  └─────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
  ┌─────────────────────────────────────────────────────────────────────────────────┐
  │                        Step 4: 位姿图优化 (g2o)                                  │
  ├─────────────────────────────────────────────────────────────────────────────────┤
  │  第225-230行: 使用 g2o 库进行非线性最小二乘优化                                    │
  │  │                                                                               │
  │  优化目标: 最小化所有观测约束的加权误差平方和                                        │
  │  │                                                                               │
  │  │   min  Σ  || z_{ij} - h(T_i, L_j) ||²_Ω_{ij}                                 │
  │  │   T,L                                                                        │
  │  │                                                                               │
  │  │   其中:                                                                       │
  │  │   ├── z_{ij}: Agent i 对 Landmark j 的观测值 (检测框位置)                      │
  │  │   ├── h(T_i, L_j): 预测函数, 将 Landmark 变换到 Agent 坐标系                  │
  │  │   ├── Ω_{ij}: 信息矩阵 (不确定性权重的倒数)                                     │
  │  │   ├── T_i: Agent i 的位姿 (待优化变量)                                        │
  │  │   └── L_j: Landmark j 的位置 (待优化变量)                                     │
  │  │                                                                               │
  │  优化过程:                                                                        │
  │  │   pgo.optimize(max_iterations=1000)                                           │
  │  │   ├── 使用 Gauss-Newton 或 Levenberg-Marquardt 方法                           │
  │  │   ├── 迭代更新所有节点的位姿估计                                               │
  │  │   └── 收敛后得到最优位姿                                                       │
  │  │                                                                               │
  │  边缘化: 优化完成后, Landmark 节点被边缘化,只保留 Agent 位姿                        │
  │  │   for agent_id in range(N):                                                   │
  │  │       refined_pose[agent_id] = pgo.get_pose(agent_id).vector()               │
  └─────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
  ┌─────────────────────────────────────────────────────────────────────────────────┐
  │                        Step 5: 返回校正后的相对位姿                               │
  ├─────────────────────────────────────────────────────────────────────────────────┤
  │  第235-240行:                                                                     │
  │  │                                                                               │
  │  ├── 提取优化后的位姿                                                              │
  │  │   refined_pose = np.array([pgo.get_pose(i).vector() for i in range(N)])      │
  │  │                                                                               │
  │  ├── 角度单位转换 (弧度 → 度)                                                       │
  │  │   refined_pose[:, 2] = rad2deg(refined_pose[:, 2])                           │
  │  │                                                                               │
  │  └── 返回结果                                                                      │
  │      return refined_pose  # [N_cav, 3] (x, y, yaw)                              │
  │                                                                                  │
  │  注意: 返回的是相对于 Ego 车的相对位姿, 只包含 x, y, yaw 三个自由度                  │
  │        其他三个自由度 (z, roll, pitch) 在地面假设下为常数                           │
  └─────────────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────────────────────────┐
  │                           box_align 配置项                                       │
  ├─────────────────────────────────────────────────────────────────────────────────┤
  │  box_align:                                                                      │
  │    enabled: true                    # 是否启用 FreeAlign                         │
  │    train_result: "stage1_boxes.json"  # Stage1 检测结果路径                      │
  │    val_result: "stage1_boxes_val.json" # 验证集检测结果路径                      │
  │    no_pose: true                    # 是否不使用外部定位 (仅用检测框校正)         │
  │    min_anchor: 4                    # 最小锚点数 (检测框数量阈值)                 │
  │    anchor_error: 0.3                # 锚框误差阈值                               │
  │    box_error: 0.5                   # 检测框误差阈值                             │
  │    gnn: false                       # 是否使用 GNN 特征提取                      │
  │    gt_correct: false                # 是否使用 GT 框进行校正 (消融实验)           │
  │    args:                            # box_alignment_relative_sample_np 参数      │
  │      thres: 1.5                     # 检测框匹配距离阈值 (米)                    │
  │      yaw_var_thres: 0.2             # 偏航角方差阈值 (弧度)                      │
  │      max_iterations: 1000           # 位姿图优化最大迭代次数                     │
  │      landmark_SE2: true             # 地标是否包含偏航角                         │
  │      adaptive_landmark: false       # 是否自适应选择地标类型                     │
  │      use_uncertainty: true          # 是否使用不确定性权重                       │
  └─────────────────────────────────────────────────────────────────────────────────┘
